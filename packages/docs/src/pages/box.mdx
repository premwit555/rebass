# Box

Responsive box-model layout component

```jsx live=true
<Box
  p={5}
  fontSize={4}
  width={[ 1, 1, 1/2 ]}
  color='white'
  bg='primary'>
  Box
</Box>
```

The Box and Flex components are also available in the standalone [Reflexbox](/reflexbox) package.
